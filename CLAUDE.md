# Package Manager
Use bun for this project. Global bun preference applies.
`mcp-oauth-provider` is Bun-only (uses `Bun.serve()` and `Bun.file()`) — Node.js is not viable.

# Branching convention
Each sub-phase or day of work gets its own branch (e.g. `day1-foundation`, `day2-core-loop`).
Never commit sub-phase work directly to main. Check git log at the start of each session to confirm the active branch convention before writing any code.
